A complete, high-performance Dental Management System designed for modern clinics. This project was originally developed as a **Final Year Project (FYP)** for a Computer Science university student by the **WorldWebTree** team in **2022**.

## ⚙️ Installation & Setup

1. **Clone the Project**:
   ```bash
   git clone https://github.com/worldwebtree/Dental-management-system.git
   cd Dental-management-system
   ```

2. **Install Dependencies**:
   ```bash
   composer install
   npm install
   ```

3. **Configure Environment**:
   - Copy `.env.example` to `.env`
   - Set your `DB_DATABASE`, `DB_USERNAME`, and `DB_PASSWORD`.

4. **Initialize Database**:
   ```bash
   php artisan key:generate
   php artisan migrate --seed
   ```

5. **Start the Application**:
   ```bash
   php artisan serve
   ```
